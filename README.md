# Фон — долой!

Небольшое нативное приложение для macOS, которое удаляет фон изображения прямо на устройстве. Оно использует системный Apple Vision: фото не загружаются в интернет, отдельная модель и Python не нужны.

## Возможности

- перетаскивание изображения в окно;
- выбор файла и вставка через буфер обмена;
- автоматическое удаление фона;
- точное автоматическое удаление однотонного насыщенного фона (green/blue screen);
- просмотр результата на прозрачном, белом или чёрном фоне;
- сравнение оригинала и результата;
- копирование результата;
- экспорт прозрачного PNG в исходном разрешении.

## Требования

- macOS 14 Sonoma или новее;
- Xcode 16 или новее — только для самостоятельной сборки.

## Запуск для разработки

```bash
swift run BackgroundAway
```

## Сборка приложения

```bash
chmod +x Scripts/build-app.sh
./Scripts/build-app.sh
open dist/BackgroundAway.app
```

Готовое приложение появится в `dist/BackgroundAway.app`. Сборка подписывается локальной ad-hoc подписью и предназначена для запуска на этом Mac. Для распространения другим пользователям понадобятся Apple Developer ID, hardened runtime и notarization.

## Проверка

```bash
swift test
```

Тест проверяет, что хромакейный фон удаляется без потери объекта и корректно обрабатываются отверстия внутри силуэта.
